| 3 | **예약 폼·목록(Day 03) + localStorage(Day 04)가 이 페이지에 통합되어 있는가?** | index.html 단일 파일 안에 #reservation 섹션, 예약 추가/삭제 폼, JS 배열 → localStorage 저장/복원, 전체 초기화 버튼 모두 통합 완료 | ✅ 완료 — 별도 파일 없이 index.html 단일 통합 |
| 4 | **빠진 내용** | ① HERO에 "픽업 예약하기" CTA 버튼 없음 ② "3~5분 픽업" 수치 명시 없음 ③ 오늘 영업 여부 실시간 표시 없음 ④ 예약 섹션에 "오늘 픽업 가능 시간대" 안내 없음 ⑤ 타이틀이 "망원동"인데 PRD는 "연남동" — 위치 불일치 확인 필요 | ❌ 개선 필요 |

## 먼저 고칠 우선순위 3가지

| 순위 | 항목 | 구체적 조치 | 근거 |
|------|------|-----------|------|
| **🥇 1순위** | HERO에 "픽업 예약하기" CTA 버튼 추가 | `<a href="#reservation" class="btn-primary">지금 픽업 예약하기</a>` 로 교체 | 첫 화면에서 핵심 행동(예약)으로 즉시 연결 — 30초 룰 미충족 해결 |
| **🥈 2순위** | "3~5분 픽업" 속도 수치 HERO 또는 배지에 명시 | HERO 배지 또는 sub 문구에 "주문 후 3~5분 내 픽업" 추가 | PRD 핵심 가치인 '빠른 픽업'을 숫자로 증명 — 신뢰감·차별화 |
| **🥉 3순위** | 예약 섹션 상단에 "오늘 픽업 가능 시간" 안내 추가 | 영업시간(08:00~20:00) 기반 안내 문구 삽입 | 방문자가 폼 작성 전 오늘 예약 가능 여부를 즉시 확인 가능 |
